# Healthcare Monitoring AI Agent

## Project Overview
Healthcare Monitoring AI Agent is an AI-based system designed to help users monitor their health and maintain healthy habits. The system collects daily health data from users and provides personalized suggestions based on their health condition.

## Problem Statement
Many people do not regularly monitor their daily health activities such as sleep, exercise, and water intake. People with medical conditions like diabetes, blood pressure, PCOS, and thyroid require regular monitoring. This project aims to provide an intelligent system that helps users track their health and receive useful health recommendations.

## Features

### 1. User Health Condition Selection
Users can select if they have any of the following conditions:
- Diabetes
- Blood Pressure (BP)
- PCOS / PCOD
- Thyroid
- No health issues

### 2. Daily Health Tracking
The system allows users to record their daily activities such as:
- Steps walked
- Sleep hours
- Water intake
- Exercise
- Symptoms

### 3. Medicine Reminder
Users can add their medicines and schedule reminders so they do not miss their medication.

### 4. AI Health Suggestions
The AI module analyzes the user's health data and provides suggestions.

Example:
- High sugar → avoid sweets
- High BP → reduce salt intake
- PCOS → increase exercise

### 5. Motivation Features
To encourage healthy habits, the system includes:
- Daily streak tracking
- Health score
- Badges for achievements
- Weekly health report

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python (Flask)

Database:
- SQLite

AI Logic:
- Rule-based AI health recommendation system using Python.

## System Workflow

1. User logs into the system.
2. User selects their health condition.
3. User enters daily health data.
4. Data is stored in the SQLite database.
5. AI module analyzes the data.
6. System provides health suggestions and reminders.

## Project Goal
The goal of this project is to develop an intelligent health assistant that helps users monitor their health, maintain good habits, and receive personalized health guidance.
