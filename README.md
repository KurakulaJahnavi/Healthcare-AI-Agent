# Healthcare Monitoring AI Agent

## Project Description
This project is an AI-based healthcare monitoring system. 
The system helps users track their daily health data, manage medicine reminders, and receive suggestions based on their health conditions such as BP, Diabetes, PCOS, etc.

## Features
- Daily health tracking
- Medication reminders
- Health condition monitoring
- AI suggestions based on user data
- Progress tracking using streaks

## Technologies Used
- Python
- Streamlit
- SQLite
- GitHub
