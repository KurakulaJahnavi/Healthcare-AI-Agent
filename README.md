# Healthcare Monitoring AI Agent

## Description
This project is an AI-based healthcare assistant that helps users track their daily health activities and provides suggestions.

## Features
- User login system
- Health condition tracking (BP, Diabetes, PCOS, etc.)
- Daily tracking:
  - Steps
  - Sleep
  - Water intake
  - Medication tracking
- Data stored using SQLite
- AI-based basic health suggestions

## Technologies Used
- Python
- SQLite

## How to Run
1. Install Python
2. Run:
   python main.py
